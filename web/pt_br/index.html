Sobre

Scala school começou como uma série de palestras no Twitter para preparar experiêntes engenheiros de software para serem produtivos programadores de Scala. Scala é uma linguagem relativamente nova, mas se baseia em muitos conceitos familiares. Assim, essas palestras assumiam que o público sabia os conceitos e mostrava como usá-los em Scala. Nós achamos que essa é uma maneira efetiva de conseguir novos engenheiros rapidamente. Este é o material escrito que acompanhava essas palestras. Nos achamos que eles são úteis por si só.

Abordagem

Nós achamos que faz mais sentido para abordagem ensinar Scala não como se fosse um Java melhorado, mas sim uma nova linguagem.Não esperamos que tenha experiência em Java. O foco será no interpretador e no estilo de objeto funcional assim como no estilo de programação que usamos aqui. Daremos uma ênfase em manutenibilidade, clareza de expressão, e alavancamento do sistema de tipos.

A maioria das lições não exige nenhum software além de um Scala REPL. O Leitor é encorajado a acompanhar as lições, mas também ir além! Use estas lições como um ponto de partida para explorar a linguagem.
