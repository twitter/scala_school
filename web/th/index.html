เกี่ยวกับเรา

โรงเรียน Scala เริ่มขึ้นจากการที่เป็นซีรีย์บรรยายภายในของ Twitter เพื่อเตรียมความพร้อมของวิศวกรให้เป็นวิศวกรภาษา Scala ที่มีประสิทธิผล. Scala เป็นภาษาโปรแกรมที่มีลักษณะใหม่ แต่ก็ยังคงแนวคิดหลายๆอย่างจากภาษาที่มีอยู่แล้วทำให้เกิดความคุ้นเคยได้ง่าย ดังนั้นเราจะถือว่าผู้ศึกษามีความรู้เรื่องแนวคิดการโปรแกรมมาก่อนแล้วแหละเราจะแสดงการในมันใน Scala เพราะเราพบว่ามันเป็นแนวทางที่มีประสิทธิภาพเพื่อจะทำให้วิศวกรเรียนรู้ได้อย่างรวดเร็ว ซึ่งเอกสารนี้ก็ใช้ประกอบในการบรรยายเหล่านั้นและเราคิดว่ามันน่าจะมีประโยชน์ในการนำไปใช้งานของแต่ละคนของผู้ศึกษาเอง

เป้าประสงค์

เราคิดว่ามันจะเข้าใจได้ดีกว่าถ้าเราไม่ได้เรียนรู้ Scala ในฐานะที่มันปรับปรุงมาจากภาษา Java แต่ให้ผู้ศึกษาคิดซะว่ามันเป็นภาษาใหม่ถอดด้ามเลย เราไม่ได้คาดหวังว่าผู้ศึกษาจะต้องผ่านประสบการณ์จาก Java มาก่อน ดังนั้นให้สนใจไปที่ตัว Interpreter และลักษณะสไตล์ object-fuctional ของภาษา Scala รวมทั้งรูปแบบการเขียนโปรแกรมตามเอกสารนี้ได้เลย โดยที่ในเอกสารนี้จะเน้นหนักไปที่ความสามารถดูแลรักษาโค้ด, ความชัดเจนของคำสั่งและการใช้ประโยชน์ของระบบชนิดตัวแปรใน Scala

ในเนื้อหาในแต่ละบทของเอกสารนั้นส่วนใหญ่ไม่ต้องใช้ซอฟต์แวร์อื่นประกอบการทำความเข้าใจ ยกเว้นแค่ Scala REPL ที่เราอยากให้คุณลองทำไปด้วยตามในเนื้อหา แต่จะลองเล่นภายหลังก็ได้ ซึ่งบทเรียนเหล่านั้นจะถูกใช้เป็นหลักแล้วอธิบายถึงตัวภาษา
